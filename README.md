## 桌面助手数据格式

### 组内信息数据格式
- 存储位置：mysql+sqlite
- 组员信息
```json
{
	authcode:"cbb003",
	group:"CBB团队",
	isroot:true,
	name:"张三",
}
```
- 组内消息
```json
{
	category: "重要",
	content: "这是消息内容",
	group: "CBB团队",
	pin: false,
	time: "2026-02-03 10:00",
	is_delete: false
}
```

### 个人任务数据格式
```json
{
	task: "这是需要完成的任务",
	time: "2026-02-03 10:00",
    status:"未完成",
    priority:"重要",
    is_delete:false
}
```
